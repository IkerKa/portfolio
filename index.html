    Astronomy

    Python N-Body Simulation

    Astronomy N-Body Simulation

    A simulation of the gravitational interaction between celestial bodies, such as planets, stars, and galaxies. The project aims to model and visualize the dynamics of these systems, providing insights into their behavior and evolution over time.

    BlueprintBorne

    Python OCR Diffusion

    BlueprintBorne: Turning Hand-Drawn Layouts into realistic Images

    A generative AI system focused on converting layout-based inputs into photorealistic compositions. The project aims to bridge the gap between schematic design and realistic visualization, ensuring that the final output respects the original spatial organization and textual cues through advanced diffusion models.

    SpatialVision

    Python Computer Vision Structure from Motion 3D Reconstruction

    Visual Place Recognition and Localization Project

    A comprehensive 3D reconstruction framework that estimates camera poses and sparse/dense cloud points from multiple 2D images of El Pilar, Zaragoza. The system implements the full SfM pipeline, including camera calibration, feature matching with LightGlue, triangulation, and Bundle Adjustment (BA), featuring a change detection module to compare historical and current visual data.

    Q-Path: Autonomous Grid Navigator

    Python Reinforcement Learning ROS Path Planning

    Q-Learning for Autonomous Navigation

    A reinforcement learning framework developed to train autonomous agents in grid-based path planning. The project implements a Q-Learning agent capable of navigating complex environments by optimizing a reward-based policy. Key features include support for 8-action movement (diagonal), environment parsing from .pgm files, and an advanced Boltzmann Exploration mechanism with noise decay to ensure a smooth transition from exploration to exploitation. It is applicable to ROS-based robotic navigation tasks, providing a foundation for real-world autonomous systems.

    The Last Arrow

    Unity C# Virtual Reality

    The Last Arrow: VR Archery Game

    A virtual reality archery game developed in Unity, where players use a bow and arrow to hit targets in a dynamic environment. The game features realistic physics-based arrow trajectories, immersive VR interactions, and three levels of increasing difficulty. Players can practice their aim and timing while enjoying a visually engaging experience.

    Paco Pill

    C++ Computer Vision Raspberry Pi Laberinth Solving

    A self-navigating robot for labyrinth solving

    We had to build from scratch a robot capable of solving a labyrinth. We implemented a computer vision system using OpenCV to detect the labyrinth's structure and a pathfinding algorithm to navigate through it. The robot was built using a Raspberry Pi, motors, and sensors, allowing it to autonomously explore and find the exit of the labyrinth.

    ArUco & Robotics

    Python Computer Vision ArUco Markers Pose Estimation

    Dynamic object localization with RGB-D data for robotic applications

    An object localization system was built using RGB-D information and ArUco markers. The system used OpenCV to detect objects within a defined plane and estimate these objects in real time thanks to these markers. Additionally, alternatives such as the use of feature matching for localization were investigated, although this was ruled out due to its instability in environments with little texture or variable lighting. This project was implemented in Python and tested in a simulated environment with Robotics Toolbox.

    Retinal Layer Segmentation

    Python Computer Vision Medical Imaging Deep Learning

    Insights into deep-learning segmentation methods of the retinal layers from Optic Coherence Tomographies

    A comparative study of deep learning-based segmentation methods for retinal layer segmentation in Optical Coherence Tomography (OCT) images. The project involved implementing and evaluating various architectures. It also included a proposal for a preprocessing pipeline to let the model focus on the relevant features of the image, improving the segmentation performance. The study provided insights into the strengths and weaknesses of each method, as well as recommendations for future research in this area.
