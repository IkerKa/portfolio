    SpatialVision

    Python Computer Vision Structure from Motion 3D Reconstruction

    Visual Place Recognition and Localization Project

    A comprehensive 3D reconstruction framework that estimates camera poses and sparse/dense cloud points from multiple 2D images of El Pilar, Zaragoza. The system implements the full SfM pipeline, including camera calibration, feature matching with LightGlue, triangulation, and Bundle Adjustment (BA), featuring a change detection module to compare historical and current visual data.

    Q-Path: Autonomous Grid Navigator

    Python Reinforcement Learning ROS Path Planning

    Q-Learning for Autonomous Navigation

    A reinforcement learning framework developed to train autonomous agents in grid-based path planning. The project implements a Q-Learning agent capable of navigating complex environments by optimizing a reward-based policy. Key features include support for 8-action movement (diagonal), environment parsing from .pgm files, and an advanced Boltzmann Exploration mechanism with noise decay to ensure a smooth transition from exploration to exploitation. It is applicable to ROS-based robotic navigation tasks, providing a foundation for real-world autonomous systems.
